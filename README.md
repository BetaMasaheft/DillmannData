# Dillmann Lexicon data

This is the data repo for use with the [Dillmann application](https://github.com/BetaMasaheft/Dillmann) to display the data resulted from the digitization process of the *Lexicon Linguae Aethiopicae* of Augustus Dillmann. It is deployed at the moment only for registered users with the relevant data at the University of Hamburg.

The data files are validated during CI.
